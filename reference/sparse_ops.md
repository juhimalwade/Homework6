# Sparse Numeric Operations

Arithmetic and utility operations for `sparse_numeric` objects.

## Usage

``` r
sparse_add(x, y, ...)

sparse_sub(x, y, ...)

sparse_mult(x, y, ...)

sparse_crossprod(x, y, ...)

standardize(x, ...)
```

## Arguments

- x:

  A `sparse_numeric` operand.

- y:

  A `sparse_numeric` operand.

- ...:

  Additional arguments (unused).
